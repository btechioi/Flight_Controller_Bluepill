## 🛸 Overview

A modular flight controller for STM32 Bluepill with individual Arduino sketches for each subsystem — IMU, GPS, barometer, compass, PID control, RTH, telemetry, and lidar.

---

## 🛠️ Modules

- **Flight_Controller** — Main loop
- **PID** — Attitude stabilization
- **IMU** — MPU6050 sensor fusion
- **GPS** — Position tracking
- **Compass** — Heading reference
- **Barometer** — Altitude hold
- **Telemetry** — Data downlink
- **RTH** — Return-to-home
- **Takeoff** — Automated launch
- **Lidar** — precision altitude
